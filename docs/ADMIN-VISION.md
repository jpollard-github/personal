# ADMIN-VISION.md

## Mission

The admin exists to help publish and maintain ArcadeGhosts.

## Core Responsibilities

- Content Inbox
- Tiny Thoughts
- Build Log
- Projects
- Home Spotlight
- Now
- Guestbook
- Social Quest Log
- Context Refresh
- Error previews

## What Belongs Here

- Publishing workflows
- Editorial tools
- Website maintenance
- Review utilities

## What Does NOT Belong Here

- Experimental playgrounds
- One-off utilities that never become part of publishing
- Framework development

## Future Direction

- Lightweight CMS
- Content-first
- Automation only where it removes repetitive work
- Not a generic CMS
