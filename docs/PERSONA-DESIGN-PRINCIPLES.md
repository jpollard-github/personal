# PERSONA-DESIGN-PRINCIPLES.md

## Core Principles

- Personas describe people.
- Archetypes describe browsing behavior.
- Scenarios describe visitor intent.
- Deterministic reports provide evidence.
- Human review is authoritative.
- Future analytics validate assumptions.
- AI interprets reports, not replaces them.
- Routes are a means, not the goal.
- Website quality is more important than individual route metrics.

## Guiding Philosophy

The framework exists to improve ArcadeGhosts.

It is not the primary product.

It exists to improve:

- website structure
- visitor understanding
- content discoverability

Content growth remains the primary long-term driver of website quality.

The framework should gradually become quieter over time. As ArcadeGhosts matures, framework changes should become less frequent while content, writing, and publishing become the primary source of improvement.

## Product Loop

```text
Website
  ↓
Persona Tests
  ↓
Journey Reports
  ↓
PERSONA-TESTS-RESULTS-TODO
  ↓
Website Improvements
  ↓

  ↓
Future Analytics
  ↓
Repeat
```
