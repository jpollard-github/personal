# Neon Forest Personal Site

A stylish personal website built with Next.js for Vercel.

## Edit Content

Most page content lives near the top of `app/page.tsx` in simple arrays:

- `projects`
- `writing`
- `music`
- `cats`
- `contactLinks`

The main visual system lives in `app/globals.css`. The generated hero image is at `public/images/neon-forest-diner.png`.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Import this folder into Vercel. The default Next.js settings are enough.
